The Nucleus Angular SASS framework include the source code from the hint.css library which provides tooltip functionality through pure CSS. This tooltip directive provides functionality to have a sticky tooltip and should only be used if you need that functionality (otherwise, the pure CSS tooltip should suffice). The position of the sticky tooltip is controlled through the data-vertical and data-horizontal attributes so you can have sticky tooltips such as top left
  • Email
  • Documents
  • Spread Sheets
 , bottom right
